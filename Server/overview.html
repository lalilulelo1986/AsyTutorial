.

SOClass™ is a Java-based system that offers an environment for managing electronic documents (e-documents), built for the virtual processing of physical documents in an organization. One e-document is represented by one application consisting of a set of Java classes, images and help files. The simplicity of development allows even beginners to build a complex and rich application that includes a high level of security. The SOClass™ application easily integrates in any existing environment through its database and operating system independence and browser compatibility. It offers easy to understand user management, an internal e-mail application and is XML compliant. E-government is facilitated through the use of the GCF™ (Government Class Framework) technology.

SOClass™ is based on a n-tier client-server architecture that supports standard clients, thin clients as well as ultra thin clients. Thin clients access the server through applets that are either installed on the client machine or downloaded through a web browser. Ultra thin clients, such as PDAs or mobile phones, access the system through pure HTML pages since Java applets are not supported on most of these ultra thin clients that lack adequate CPU power.

SOClass™ is built on 100% pure java (J2SE v.1.4) and is therefore platform independent.
The integrated PKI (Public key infrastructure) based encryption - 1024bit Key - and SSL guarantee high security to the system and its users. The connection to popular databases is assured through java's JDBC (Java Database connector).

Embedded services
The SOClass™ environment offers in particular the following integrated services:
Email - exchange emails with other SOClass™ users in your network, with attached e-documents
User administration - define business units, user profiles, access privileges etc.
Document history - retrace every state of your e-document
Document library - navigate easily through your e-documents
Templates - use these to create quickly a new document
MyProfile - adjust your preferences according to preferred languages, locales
Finder - powerful search engine
Help - HTML help files
